## What I Work On

**Speech & TTS**
Trained Tacotron 1 and Tacotron 2 with WaveGlow for Punjabi Shahmukhi
and Urdu from scratch. Fine-tuned Facebook MMS VITS for a low-resource
language. Published at Interspeech 2025.

**LLM & RAG Systems**
Built agentic RAG pipelines with LangGraph, hybrid vector search, query
decomposition, and intent routing. Experience with AWS Bedrock, OpenSearch
Serverless, Weaviate, and Claude via the Anthropic API.

**Data & Analytics**
PySpark pipelines on Microsoft Fabric for retail analytics including
store clustering, replenishment classification, and inventory dashboards.

**AI Agents & Automation**
Built Claude-based agents and skills using Claude Code, including a morning
standup tool that scans project folders and auto-generates a daily briefing.
